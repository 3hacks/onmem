# onmem
onmem is data storage on memory. You should think it as a clipboard.

### Requirements
* Ubuntu 20.10
* Node.js

### Usage
**install**
```bash
npm install -g TakutoYoshikai/onmem
```

**run process**
```bash
onmem-run
```

**set data**
```bash
echo <data> | onmem-post
```

**get data**
```bash
onmem
```
